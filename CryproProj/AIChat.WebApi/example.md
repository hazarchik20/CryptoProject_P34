## Що таке C#?

### 1. Визначення

**C# (C sharp)** – це об’єктно‑орієнтована мова програмування, створена компанією Microsoft у 2000‑х роках як частина платформи **.NET**. Вона поєднує простий, читабельний синтаксис (подібний до Java чи C++) і потужний набір бібліотек, що дозволяє розробляти:

- веб‑додатки,
- настільні програми,
- мобільні застосунки,
- ігри,
- мікросервіси,
- і багато іншого.

### 2. Порівняння з реальним світом

Уявіть **C#** як **швейцарський ніж**:

| Швейцарський ніж | C# |
|-------------------|----|
| **Одна рукоятка** — зручний інтерфейс | **Чіткий синтаксис** — легко читати і писати |
| **Різні леза** — інструменти для різних задач | **Багаті бібліотеки .NET** — робота з БД, мережею, графікою, безпекою тощо |
| **Працює майже будь‑де** | **Крос‑платформений** (Windows, Linux, macOS) завдяки .NET Core/.NET 5+ |

Як і швейцарський ніж, C# підходить для багатьох задач без потреби в окремих інструментах.

### 3. Мінімальний приклад коду

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Привіт, світ!");
    }
}
```

- `using System;` – підключає простір імен з базовими типами (наприклад, `Console`).
- `class Program` – визначає клас, у якому розміщений код.
- `static void Main()` – точка входу в програму; виконується першою.
- `Console.WriteLine` – виводить текст у консоль.

**Результат:** після запуску програма виведе *«Привіт, світ!»*.

### 4. Де C# використовується у реальних проектах

| Сфера | Приклад використання |
|-------|----------------------|
| **Веб‑розробка** | ASP.NET Core – масштабовані веб‑API та сайти (наприклад, Stack Overflow). |
| **Настільні програми** | Windows Forms, WPF – корпоративні CRM‑системи, утиліти. |
| **Мобільні додатки** | Xamarin / .NET MAUI – крос‑платформені iOS та Android програми. |
| **Ігрова індустрія** | Unity Engine – більшість інді‑ігор та частина AAA‑проектів. |
| **Мікросервіси** | .NET 6/7 + Docker, Azure Functions – розподілені сервіси в хмарі. |

#### Чому компанії обирають C#?

- **Висока продуктивність** завдяки JIT‑компіляції та оптимізації CLR.
- **Мультиплатформеність** – один код запускається на різних ОС.
- **Сильна типізація** → менше помилок під час компіляції.
- **Асинхронне програмування (`async/await`)** – просте створення масштабованих, неблокуючих додатків.
- **Багата екосистема** – готові бібліотеки, інструменти, документація.
